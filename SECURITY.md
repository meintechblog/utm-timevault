# Security Policy

## Supported Versions

Security fixes are provided for the latest release on `main`.

## Reporting a Vulnerability

Please do not open public issues for sensitive vulnerabilities.

Report privately by contacting the maintainer via GitHub profile: `meintechblog`.
Include:

- Affected version/commit
- Reproduction steps
- Impact assessment
- Suggested mitigation (if available)

We will acknowledge receipt and provide a remediation timeline.
